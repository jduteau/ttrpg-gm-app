# Dragonbane Campaign State Fields

Append these fields inside the SESSION STATE block, after the universal fields:

```
## Conditions & Injuries
[List any active conditions or injuries per PC: cold/scared/angry/hungry/thirsty/exhausted]

## Willpower
[Current WP per PC if relevant]

## Adventure Site
area: [current area of the adventure site if in one]
hazards_known: [traps or hazards the party has identified]
explored: [rooms or areas cleared]

## Dragon
dragon_active: [yes/no — is the dragon a current active threat]
dragon_notes: [any relevant dragon state if applicable]
```
